# <a name="power-bi-premium-faq"></a>Power BI Premiumin usein kysytyt kysymykset
Selaa Power BI Premium -tarjoomaa koskevien usein kysyttyjen kysymysten ja vastausten luetteloa.

* Jos sinulla on muita kysymyksiä, [voit esittää niitä Power BI -yhteisössä](http://community.powerbi.com/).
* Eikö ongelma ratkennut? Käy [Power BI -tukisivulla](https://powerbi.microsoft.com/support/).
* Voit myös valita tämän selainikkunan oikeassa alakulmassa **Ottakaa minuun yhteyttä** -kuvakkeen ![](media/service-premium-faq/power-bi-contact-me-icon.png).

## <a name="power-bi-premium"></a>Power BI Premium
**Mikä on Power BI Premium?**  
Power BI Premium on uusi kapasiteettipohjainen tarjooma, joka tarjoaa:

* mahdollisuuden julkaista raportteja yrityksenlaajuisesti ilman, että vastaanottajilla on oltava yksilöllinen käyttöoikeus.
* paremman skaalautuvuuden ja suorituskyvyn erillisestä kapasiteetista Power BI -palvelussa.
* mahdollisuuden ylläpitää BI-resursseja paikallisesti Power BI -raporttipalvelimen avulla.
* yhden ohjelmointirajapinnan, yhtenäisen ominaisuusjoukon ja pääsyn uusimpiin ominaisuuksiin upotettuja analyysitoimintoja varten.

**Mitä Power BI Premium tekee? Miten se toimii?**  
Power BI Premium koostuu Power BI -palvelun kapasiteetista, joka on pelkästään varattu kullekin organisaatiolle ja jota Microsoftin täysin hallinnoima erillinen laitteisto tukee. Organisaatiot voivat käyttää erillisen kapasiteettinsa laaja-alaisesti tai varata sen määritetyille työtiloille käyttäjien määrän, kuormituksen tai muiden tekijöiden perusteella sekä lisätä tai vähentää sitä vaatimusten muuttuessa.

**Miten Power BI Pro poikkeaa Power BI Premiumista? Miksi Power BI Premium on tarpeen?**  
Power BI Premium on kapasiteettipohjainen käyttöoikeus, kun taas Power BI Pro on käyttäjäpohjainen käyttöoikeus.

Power BI Premium koostuu Power BI -palvelun kapasiteetista, joka on pelkästään varattu kullekin organisaatiolle ja jota Microsoftin täysin hallinnoima erillinen laitteisto tukee. Se poistaa tietojoukkojen koon ja päivittämistiheyden rajoitukset ja tukee eristämistä erillisen laitteiston ansiosta. Power BI Premium tarjoaa myös mahdollisuuden julkaista raportteja yrityksenlaajuisesti ilman, että vastaanottajilla on oltava yksilöllinen käyttöoikeus.

Power BI Pro on tarkoitettu käyttäjille, jotka julkaisevat raportteja, jakavat raporttinäkymiä, tekevät yhteistyötä työtovereiden kanssa työtiloissa ja käyttävät muita aiheeseen liittyviä toimintoja, kuten:

* mukautettujen näkymien muokkaus ja tallentaminen
* omien raporttinäkymien luominen (kiinnittäminen uuteen raporttinäkymään)
* tietojen analysointi Excelissä tai Power BI Desktopissa
* jakaminen Excel Web App -tuen ansiosta
* koontinäyttöjen jakaminen ja yhteistyö Office 365 -ryhmien kanssa
* Integroi sisältöä Microsoft Teamsin kanssa

**Tarvitsenko Power BI Pron Power BI Premiumin käyttämiseen?**  
Kyllä. Power BI Prolla voit julkaista raportteja, jakaa raporttinäkymiä, tehdä yhteistyötä työtovereiden kanssa työtiloissa ja käyttää muita aiheeseen liittyviä toimintoja.

**Kuinka paljon Power BI Premium maksaa? Kuinka monta varastointiyksikköä on käytettävissä?**  
Power BI Premiumin voi hankkia näennäisytimien määrän perusteella. Voit tarkistaa hinnat [Power BI:n hinnoittelusivulta](https://powerbi.microsoft.com/pricing/). Katso lisätietoja solmuista ja näennäisytimistä [Microsoft Power BI Premiumin teknisestä raportista](https://aka.ms/pbipremiumwhitepaper). Lisäksi tämän [laskurin](https://powerbi.microsoft.com/calculator/) avulla voit laskea, kuinka paljon Premium-kapasiteettia suunnilleen tarvitset.

**Mitä kapasiteetilla tarkoitetaan?**  
Kapasiteetti viittaa nimettyyn kapasiteettiin, jonka järjestelmänvalvoja on valmistellut Power BI Premiumin kapasiteetin hallintaportaalin kautta. Kapasiteetti on yhdestä tai useammasta solmusta koostuva ryhmittely. Katso lisätietoja [Microsoft Power BI Premiumin teknisestä raportista](https://aka.ms/pbipremiumwhitepaper).

**Miten Power BI Premium laskutetaan?**  
Power BI Premium laskutetaan kuukausittain vuositilauksen pohjalta.

**Miten Power BI Premiumin voi ostaa?**  
Power BI Premium on saatavilla Power BI Pron lisäosana O365-hallintakeskuksessa. Lisätietoja on artikkelissa [Ohjeet Power BI Premiumin ostamiseen](service-admin-premium-purchase.md). Voit myös ottaa yhteyttä Microsoftin edustajaan ja kysyä lisätietoja.

**Mistä tietää, kuinka paljon Power BI Premium -kapasiteettia on hankittava?**  
Tämän [kustannuslaskurin](https://powerbi.microsoft.com/calculator/) avulla voit laskea, kuinka paljon Power BI Premium -kapasiteettia suunnilleen tarvitset.

**Milloin Power BI Premium on saatavilla? Onko se saatavilla kaikilla alueilla samanaikaisesti?**  
Power BI Premium tulee yleisesti saataville vuoden 2017 toisen vuosineljänneksen lopussa niille alueille, jotka tällä hetkellä tukevat Power BI -palvelua.

**Täyttääkö Power BI Premium julkaisuhetkellä sertifiointien ja säännösten vaatimukset? Entä kansalliset pilvipalvelut?**  
Power BI Premium tulee yleisesti saataville vuoden 2017 toisen vuosineljänneksen lopussa niille alueille, jotka tällä hetkellä tukevat Power BI -palvelua. Kansalliset pilvipalvelut (tarkemmin Saksa, Kiina ja Yhdysvaltain valtionhallinto) tulevat saataville todennäköisesti vuoden 2017 kolmannen vuosineljänneksen aikana. Power BI:n sertifiointi- ja säännöstenmukaisuusstandardeja sovelletaan Power BI Premiumiin, kun se tulee yleisesti saataville.

**Onko Power BI Premium saatavilla Office 365 E5:n kautta?**  
Power BI Premium on saatavilla Power BI Pron lisäosana. Office 365 E5 sisältää Power BI Pron. E5-asiakkaat voivat ostaa Premiumin lisäosana heidän nykyisiin Pro-käyttöoikeuksiinsa.

**Mikä on Power BI -raporttipalvelin? Tarkoittaako tämä, että Power BI on käytettävissä paikallisesti?**  
Power BI -raporttipalvelin on saatavilla osana Power BI Premiumia. Se on yhteensopiva SQL Server Reporting Services -raporttien kanssa, mikä mahdollistaa vuorovaikutteisten Power BI -raporttien sekä sivutettujen raporttien käyttöönoton ja jakamisen paikallisesti organisaation palomuurin rajojen puitteissa. Organisaatiot voivat valita Power BI:n pilvipalveluversion tai säilyttää raportit paikallisesti Power BI -raporttipalvelimella ja siirtää tiedot pilvipalveluun omassa tahdissaan.

Power BI -ominaisuudet, kuten reaaliaikaiset raporttinäkymät, luonnollisen kielen kyselyt ja muut ominaisuudet Power BI -palvelussa, eivät ole käytettävissä osana Power BI -raporttipalvelinta. 

Lisätietoja Power BI -raporttipalvelimesta on artikkelissa [Power BI -raporttipalvelimen käytön aloittaminen](report-server/get-started.md).

**Onko Power BI -raporttipalvelin sama ominaisuus, jonka Microsoft tarjosi SSRS:n teknisessä esiversiossa Power BI -raporttien nimellä? Miksi tämä on nyt osa Power BI:tä, kun se vaikutti SQL:n ominaisuudelta?**  
Power BI -raporttipalvelin on saatavilla osana Power BI Premiumia. Se on yhteensopiva SQL Server Reporting Services -raporttien kanssa, mikä mahdollistaa vuorovaikutteisten Power BI -raporttien sekä sivutettujen raporttien käyttöönoton ja jakamisen paikallisesti organisaation palomuurin rajojen puitteissa.

Ilmoitus Power BI Premiumin julkaisemisesta oli ensimmäinen kerta, kun kerroimme, miten tämä toiminto tuodaan markkinoille.

Lisätietoja Power BI -raporttipalvelimesta on artikkelissa [Power BI -raporttipalvelimen käytön aloittaminen](report-server/get-started.md).

**Mitä ovat Power BI -sovellukset?**  
Kehitämme Power BI -sovelluksiin sisältöpaketteja, jotta käyttäjien on helpompi löytää merkityksellisiä tietoja ja tutustua niihin yritystasolla. Nyt saatavilla olevat Power BI -sovellukset tarjoavat yksinkertaistetun tavan ottaa käyttöön raporttinäkymiä ja raportteja tietyille henkilöille, ryhmille tai koko organisaatiolle. Yrityskäyttäjät voivat asentaa nämä sovellukset ja käyttää niitä helposti sekä keskittää sisällön yhteen paikkaan ja päivittää niitä automaattisesti. 

Lisätietoja sovelluksista on artikkelissa [Mitä ovat Power BI:n sovellukset?](service-install-use-apps.md).

**Onko Power BI Prohon, Power BI:n ilmaisversioon tai Power BI Desktopiin tulossa muutoksia Power BI Premiumin käyttöönoton myötä?**  
Nyt teemme myös selkeämmän eron Power BI Pron ja maksuttoman palvelun välille. Olemme saaneet palautetta, että vaikka maksuton palvelu on tarkoitettu henkilökohtaiseen käyttöön ja Power BI Pro yhteistyöhön, toiminnalliset eroavaisuudet ovat aiheuttaneet hämmennystä käyttäjien joukossa. Jatkossa parannamme maksutonta palvelua siten, että siinä on samat toiminnot kuin Power BI Prossa mutta jakamis- ja yhteistyöominaisuudet ovat vain Power BI Pro -käyttäjien käytettävissä.

Maksuttoman Power BI -palvelun käyttäjät hyötyvät kaikkien tietolähteiden käyttömahdollisuudesta, suuremmista työtilan tallennusrajoista sekä suuremmista päivitys- ja virtautusnopeuksista. Nämä muutokset tulevat voimaan 1.6.2017. Lisätietoja on artikkelissa [Omatoiminen kirjautuminen Power BI:hin](service-self-service-signup-for-power-bi.md). Power BI Desktop on edelleen saatavilla maksutta.

Lisätietoja laajennetusta Pro-kokeiluversion tarjoomasta on artikkelissa [Laajennetun Pro-kokeiluversion aktivointi](service-extended-pro-trial.md).

## <a name="power-bi-free"></a>Power BI (maksuton)
**Mitä ominaisuuksia maksuttomaan palveluun lisätään?**  
1.6.2017 alkaen maksuttomassa palvelussa on Power BI Prota vastaavat ominaisuudet lukuun ottamatta jakamista, yritystason jakelua ja yhteistyötä. Siihen sisältyvät sama 10 Gt:n työtilarajoitus, enintään kahdeksan päivittäistä ajoitettua tietojoukkojen päivitystä ja tietojen virtautuksen enimmäisnopeus 1 M/tunti. Lisäämme myös pääsyn kaikkiin tietolähteisiin, myös niihin, jotka ovat käytettävissä paikallisen tietoyhdyskäytävän kautta.

**Mitkä Power BI Pro -ominaisuudet eivät ole käytettävissä maksuttomassa palvelussa?**  
Vertaisten välinen raporttinäkymän jakaminen, sovelluksen työtilat (aiemmin ryhmän työtilat), analysointi Excelissä ja Power BI -sovelluksissa (ellei Premium-kapasiteetin tukema) ovat ominaisuuksia, jotka ovat vain Power BI Prossa.

**Milloin nämä muutokset tulevat voimaan?**  
Maksuttoman palvelun muutokset tulevat voimaan 1.6.2017.

**Miten maksuttoman palvelun käyttäjille ilmoitetaan näistä muutoksista?**  
Maksuttoman palvelun aktiiviset käyttäjät, joihin 1.6.2017 voimaan astuvat muutokset vaikuttavat, saavat Microsoftilta sähköpostiviestin sekä ilmoituksen, kun he kirjautuvat sisään Power BI -palveluun 3.5.2017 alkaen.

Maksuttoman palvelun käyttäjät, jotka ovat olleet aktiivisia kuluneen vuoden aikana, saavat 12 kuukauden laajennetun Power BI Pro -kokeiluversion ilmaiseksi. Kokeiluversio tarjoaa Power BI Pron täydet ominaisuudet, jotta käyttäjillä on riittävästi aikaa sopeutua palvelun käyttöön. Lisätietoja laajennetusta Pro-kokeiluversion tarjoomasta on artikkelissa [Laajennetun Pro-kokeiluversion aktivointi](service-extended-pro-trial.md).

**Menettävätkö käyttäjät silloin pääsyn sisältöönsä (raporttinäkymiin ja raportteihin)?**  
Nro Käyttäjät eivät menetä pääsyä sisältöön, jonka he ovat ladanneet Power BI -palveluun. Muiden kanssa jaettu sisältö ei ehkä ole enää vastaanottajien käytettävissä 1.6.2017 alkaen. Samalla tavalla heillä ei ehkä ole enää pääsyä sisältöön, jonka muut ovat jakaneet heidän kanssaan.

Maksuttoman palvelun käyttäjät, jotka ovat olleet aktiivisia kuluneen vuoden aikana, saavat 12 kuukauden laajennetun Power BI Pro -kokeiluversion ilmaiseksi. Kokeiluversio tarjoaa Power BI Pron täydet ominaisuudet, jotta käyttäjillä on riittävästi aikaa sopeutua palvelun käyttöön. Lisätietoja laajennetusta Pro-kokeiluversion tarjoomasta on artikkelissa [Laajennetun Pro-kokeiluversion aktivointi](service-extended-pro-trial.md).

**Mistä käyttäjät tietävät, saavatko he käyttöönsä laajennetun Power BI Pro -kokeiluversion?**  
Katso lisätietoja laajennettua Pro-kokeiluversiota koskevista [ehdoista ja määräyksistä](https://aka.ms/power-bi-trial).

Lisätietoja laajennetusta Pro-kokeiluversion tarjoomasta on artikkelissa [Laajennetun Pro-kokeiluversion aktivointi](service-extended-pro-trial.md).

**Mitä käyttäjien pitäisi tehdä, jos he eivät saa käyttöönsä laajennettua Power BI Pro -kokeiluversiota?**  
Rekisteröidy 60 päivän Power BI Pro -peruskokeiluversion käyttäjäksi [täällä](https://app.powerbi.com/signupredirect?pbi_source=web).

**Muuttuvatko maksuttoman palvelun ominaisuudet Power BI Premiumin myötä?**  
Maksuttoman palvelun käyttäjät, joilla on erillisen kapasiteetin käyttöoikeus Power BI Premiumissa, voivat vastaanottaa sisältöä, jonka Power BI Pro -käyttäjät ovat jakaneet heille julkaistujen sovellusten kautta.

## <a name="power-bi-pro"></a>Power BI Pro
**Mikä Power BI Pro -versiossa muuttuu?**  
Power BI Pro -versioon ei ole tulossa muutoksia.

**Entä Office, sisältyykö Power BI Pro edelleen Office 365 E5:een?**  
Kyllä. Power BI Pro sisältyy jatkossakin Office 365 E5:een.

**Voinko palauttaa Power BI Pro -käyttöoikeuteni, jos haluan vaihtaa Power BI Premiumiin?**  
Nro Power BI Premium on Power BI Pron lisäosa, minkä vuoksi Premiumin käyttö edellyttää Pro-käyttöoikeutta.

**Miksi Power BI Pro on tarpeen? Eikö minun vain kannata hankkia Power BI Premium?**  
Power BI Premium on kapasiteettipohjainen käyttöoikeus, kun taas Power BI Pro on käyttäjäpohjainen käyttöoikeus. Se on tarkoitettu nopeaan ja helppokäyttöiseen omatoimiseen analysointiin sellaisille käyttäjille, jotka tarvitsevat yhteistyöominaisuudet, raporttinäkymien jakamismahdollisuuden, ad-hoc-analyyseja ja mahdollisuuden julkaista raportteja, kuten:

* mukautettujen näkymien muokkaus ja tallentaminen
* omien raporttinäkymien luominen (kiinnittäminen uuteen raporttinäkymään)
* tietojen analysointi Excelissä tai Power BI Desktopissa
* Excel-työkirjojen jakaminen ja tarkastelu Excel Web App -tuen avulla
* koontinäyttöjen jakaminen ja yhteistyö Office 365 -ryhmien kanssa
* Integroi sisältöä Microsoft Teamsin kanssa

## <a name="power-bi-embedded"></a>Power BI Embedded
**Mikä on muuttunut Power BI Embedded -versiossa?**  
Osana Power BI Premiumia olemme yhdistämässä Power BI Embedded -version Power BI -palvelun kanssa, jotta voisimme tarjota yhden ohjelmointirajapinnan, yhtenäisen ominaisuusjoukon ja pääsyn uusimpiin ominaisuuksiin. Jatkossa niiden, jotka ovat kiinnostuneita upottamaan Power BI:n sovelluksiinsa, kannattaa aloittaa Power BI Desktopilla ja suorittaa käyttöönotto Power BI Premiumilla.

**Millaisia hyötyjä Power BI Premium tarjoaa analyysien upottamiseen sovelluksiin?**  
Power BI Premium on kapasiteettipohjainen tarjooma, joka suoritetaan erillisessä laitteistossa. Osana Power BI -palvelua se on täysin Microsoftin hallinnoima. Sovelluskehityksen elinkaarta ajatellen Power BI Premium tarjoaa uudenlaisia etuja, kuten joustavat ja ennustettavat kustannukset, parannetun sisällönhallinnan kehittäjille ja täydet Power BI -ominaisuudet upotettuja skenaarioita varten.

**Onko minun siirrettävä nykyinen sovellukseni, joka pohjautuu Power BI Embedded -versioon?**  
Power BI Embedded on edelleenkin olemassa olevien sovellusten saatavilla seuraavan aikataulun mukaisesti:

| Käyttöoikeussopimus | Power BI Embedded -version saatavuus |
| --- | --- |
| Enterprise Agreement -sopimus |Kunnes nykyinen sopimus päättyy |
| Suora ja CSP |Vuosi siitä, kun Power BI Premium tulee yleisesti saataville |

Kyllä, sinun kannattaa harkita upotettujen analyysitoimintojen ratkaisun siirtämistä Power BI Premiumiin. Katso lisätietoja artikkelista [Miten Power BI Embedded -työtilakokoelman sisältö siirretään Power BI -palveluun](developer/migrate-from-powerbi-embedded.md).

**Miten siirrän nykyisen sovellukseni, joka pohjautuu Power BI Embedded -versioon?**  
Katso lisätietoja sovellusten siirtämisestä [siirtoa koskevista ohjeista](developer/migrate-from-powerbi-embedded.md).

**Aikooko Microsoft pettää maksutonta Power BI Embedded -versiota koskevat lupauksensa?**  
Edellisessä mallissa tarjosimme rajoitetun määrän istuntoja kehittämiseen ja testaukseen. Uudessa skenaariossa asiakkailla on edelleen mahdollisuus aloittaa tuotantovalmiuteen tähtäävä kehitystyö ja testaus maksutta. Asiakkaat pääsevät alkuun kehityksessä ja testauksessa kahden kuukauden Power BI Pro -kokeiluversion avulla. Tietojen selaustyökalu ja raporttien luontityökalu Power BI Desktop on jatkossakin saatavilla maksutta.

**Onko Power BI Embedded edelleen Azure-palvelu? Missä voin ostaa ja valmistella Power BI Embedded -version ja hallinnoida sitä?**  
Power BI Embedded yhdistyy Power BI -palveluun, minkä vuoksi se hankitaan ja valmistellaan ja sitä hallinnoidaan jatkossa tämän mukaisesti. Lisätietoja on artikkelissa [Upottaminen Power BI:llä](developer/embedding.md).

## <a name="power-bi-report-server"></a>Power BI -raporttipalvelin
**Miksi Power BI -raporttipalvelin sisältyy Power BI Premiumiin sen sijaan, että sitä myytäisiin erillisenä tuotteena?**  
Power BI Premium tarjoaa mahdollisuuden ylläpitää BI-resursseja paikallisesti Power BI -raporttipalvelimen avulla. Power BI -raporttipalvelin on paikallinen palvelin, joka mahdollistaa vuorovaikutteisten Power BI -raporttien ja sivutettujen raporttien käyttöönoton ja jakamisen täysin organisaation palomuurin puitteissa. Power BI Premiumin avulla sama määrä organisaation pilvipalveluun valmistelemia näennäisytimiä voidaan ottaa käyttöön myös paikallisesti Power BI -raporttipalvelimen kautta ilman, että kapasiteettia tarvitsee jakaa. Organisaatiot voivat valita Power BI:n pilvipalveluversion tai säilyttää raportit paikallisesti Power BI -raporttipalvelimella ja siirtää tiedot pilvipalveluun omassa tahdissaan.

Lisätietoja on artikkelissa [Power BI -raporttipalvelimen käytön aloittaminen](report-server/get-started.md).

**Miten asiakkaat voivat käyttää Power BI -raporttipalvelinta?**  
Power BI -raporttipalvelin on käytettävissä Power BI Premiumin kautta tai etuna asiakkaille, joilla on SQL Server EE ja SA. Kysy lisätietoja Microsoftin edustajaltasi.

**Miksi Power BI -raporttipalvelin ei ole saatavilla etuna SQL Server Standard- tai Enterprise-versiota käyttäville asiakkaille?**  
Power BI -raporttipalvelin on sellaisten SQL Server -asiakkaiden käytettävissä, joilla on EE- ja SA-käyttöoikeudet. Se on myös ostettavissa Power BI Premiumin kautta. Power BI on SaaS-palvelu, ja Power BI -raporttipalvelin noudattaa samaa tilauspohjaista toimitusmallia.
