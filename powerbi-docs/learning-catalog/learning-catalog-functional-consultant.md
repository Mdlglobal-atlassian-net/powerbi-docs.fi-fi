# <a name="functional-application-consultant-learning-catalog"></a>Toiminnallisen sovelluskonsultin oppimishakemisto

Oletko yrityksen toimialueen toteutusasiantuntija? 

Seuraava hakemisto on järjestetty perustiedosta tiettyihin toimialueisiin ja perustasosta kehittyneimpään tasoon. Jos sisältöä on olemassa useissa muodoissa, kerromme sinulle siitä, jotta voit valita tarpeitasi parhaiten vastaavan opetusmuodon.

Funktionaalisten konsulttien työtehtävien analysointitutkimus on osoittanut, että kaikkien on ymmärrettävä sekä määritelty ydinsisältö että toimialueet.
